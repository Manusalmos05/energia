# Proyecto de Machine Learning - Energía

## Estructura de Carpetas

- `data/raw/`: Datos originales sin procesar.
- `data/processed/`: Datos procesados y listos para análisis/modelado.
- `notebooks/`: Jupyter Notebooks para exploración y experimentos.
- `models/`: Modelos entrenados y scripts relacionados.
- `api/`: Implementación de la API con Streamlit.
- `docs/`: Documentación adicional del proyecto.

## Requisitos
Ver `requirements.txt` para dependencias.

## Ignorar
Ver `.gitignore` para archivos y carpetas excluidas del control de versiones.
